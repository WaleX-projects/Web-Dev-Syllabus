# Web Dev Syllabus

A beautifully designed, interactive web development learning syllabus that tracks your progress across HTML, CSS, and JavaScript fundamentals.

## 📋 Overview

**Web Dev Syllabus** is an interactive progress tracker for mastering core web development technologies. It provides a structured curriculum covering essential topics in HTML5, CSS, and JavaScript, complete with a visual progress indicator and persistent local storage to track your learning journey.

## ✨ Features

- **Organized Learning Path**: Comprehensive curriculum divided into three core areas:
  - HTML (10 topics)
  - CSS (11 topics)
  - JavaScript (12 topics)

- **Progress Tracking**: Visual progress bar that updates in real-time as you complete learning objectives

- **Tab-Based Navigation**: Switch between HTML, CSS, and JavaScript modules seamlessly

- **Interactive Checklist**: Mark topics as completed with persistent storage using browser localStorage

- **Responsive Design**: Works seamlessly on desktop and mobile devices

- **Modern UI**: Clean, minimalist interface with smooth animations and intuitive interactions

## 🚀 Getting Started

1. **View the Syllabus**: Open `index.html` in your web browser
2. **Start Learning**: Navigate through HTML, CSS, and JavaScript tabs
3. **Track Progress**: Check off topics as you complete them
4. **Save Progress**: Your progress is automatically saved in browser storage

### No Installation Required
This is a standalone HTML file with no external dependencies. Simply open it in any modern web browser.

## 💻 Technologies Used

- **HTML5**: Semantic structure
- **CSS3**: Modern styling with CSS variables, Flexbox, and animations
- **JavaScript (Vanilla)**: DOM manipulation and localStorage API

## 💾 How Progress is Saved

Your learning progress is automatically saved to your browser's localStorage as `lightProgress`. This means:
- Your progress persists across browser sessions
- Each browser/device has its own progress tracking
- Clearing browser data will reset your progress

## 🎨 Design Features

- **Color-Coded Topics**: Visual tags identify which technology each topic covers
- **Progress Bar**: Rainbow gradient progress indicator showing overall completion percentage
- **Smooth Interactions**: Hover effects and smooth transitions for better UX
- **Mobile Responsive**: Adapts to various screen sizes

## 📖 Usage Tips

1. **Learn Sequentially**: Work through topics in order for best learning outcomes
2. **Track Consistently**: Regularly check off completed topics to maintain motivation
3. **Cross-Reference**: Topics build on each other—revisit earlier topics if needed
4. **Practice Projects**: After each section, apply your knowledge in real projects
